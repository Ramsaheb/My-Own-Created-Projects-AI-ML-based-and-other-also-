Fake News Detector

How to Use

  1. Select text on any webpage AND right-click on an image
  2. Choose "Detect Fake News" from the context menu
  3. View the analysis results in the notification

Important: Both text selection AND image are required for analysis to work!

This extension helps you identify potentially fake news by analyzing text and images using advanced AI techniques.

The extension communicates with a backend API running at http://localhost:8000 to perform the analysis.

Note: Make sure the backend server is running before using this extension.
